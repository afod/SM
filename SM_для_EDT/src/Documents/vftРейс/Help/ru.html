Создание рейса

Для создания нового рейса в системе необходимо выполнить одну из последовательностей действий, указанных на рисунке 5.

Рисунок 5. Создание нового рейса

Варианты создания нового рейса в базе данных:

1.       Раздел «РДО» -> переключатель списка «Рейсы» в левой части экрана -> команда «создать» на командной панели списка всех рейсов

2.       Раздел «РДО» -> команда «Рейс» в меню «Создать»

 

Для нового рейса обязательно необходимо указать следующие параметры (см. рис.6):

·         дату начала рейса (1),

·         дату окончания рейса (2),

·         порты отправления (3),

·         порт погрузки (4),

·         порт выгрузки (5),

·         груз (6)

Остальные параметры рейса (Осадка, Загрузка, Время маневров в море, время работы котлов, время подогрева грузов, время работы дизель-генераторов, капитана и механика) можно заполнить по завершении рейса. Эти данные необходимы для автоматического заполнения рейсового отчета.

 

Рисунок 6. Форма рейса

Для выбора значения в какое-либо из вышеперечисленных полей необходимо нажать многоточие в поле ввода значения (см. рис. 7)

Рисунок 7. Выбор значения поля

В отрывшемся списке найти необходимое значение, выбрать его двойным щелком мыши или «Встать» на него указателем мыши и нажать кнопку «Выбрать».

При выборе портов погрузки/выгрузки есть возможность указать несколько портов. Для этого при подборе значения необходимо выбрать не пункт следования, а строку и ввести значение вручную.

Рисунок 8. Выбор значения пункта следования

После заполнения необходимых параметров (действие 7 рис. 6) рейс необходимо записать в систему (кнопка «Провести и закрыть»).

После ввода нового рейса его необходимо выбрать в форме путевого журнала (рис. 9). В случае, если рейс был выбран на форме путевого журнала ранее, он сохраняется до следующего открытия формы автоматически.
